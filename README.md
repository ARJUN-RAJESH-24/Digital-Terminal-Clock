# Digital-Terminal-Clock
## A digital terminal clock made using C.
- can be using in windows and linux by just changing header file
-  use <unistd.h> for linux and <Windows.h> for windows

### compile using
- gcc DigitalClock.c -o Digital clock
### Headers used:
- stdio.h
- time.h
- unistd.h
- windows.h
- stdbool.h
